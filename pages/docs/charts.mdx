# Charts

## Apex Chart:

[ApexCharts](https://apexcharts.com/) is a modern JavaScript library for creating interactive charts and visualizations with an easy-to-use API. For detailed instructions and additional options, please consult the [Apex Charts](https://apexcharts.com/) [Documentation](https://apexcharts.com/docs/installation/).

## Re Chart:

[Recharts](https://recharts.org/en-US/) - Re-designed charting library built with React and D3. For detailed instructions and additional options, please consult the [Recharts](https://recharts.org/en-US/) [Documentation](https://recharts.org/en-US/guide/installation).

## Re Chart:

[Recharts](https://www.chartjs.org/)is a free, open-source JavaScript library for data visualization, which supports eight chart types: bar, line, area, pie, bubble, radar, polar, and scatter. For detailed instructions and additional options, please consult the [Recharts](https://www.chartjs.org/) [Documentation](https://www.chartjs.org/docs/latest/).

## Unovis:

[Unovis](https://unovis.dev/) is built with Typescript and allows you to import individual component modules to reduce your app bundle size. For detailed instructions and additional options, please consult the [Unovis](https://unovis.dev/docs/intro/) [Documentation](https://unovis.dev/docs/intro/).
